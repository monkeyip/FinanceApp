# 家庭资产结构体检工具（Family Asset Checkup）

一个面向普通家庭的 **家庭资产结构分析与体检工具**，  
帮助用户快速了解家庭资产分布、流动性状况及潜在结构风险。

> 本工具仅用于家庭资产结构分析与学习交流，不构成任何投资建议。

---

## ✨ 项目介绍

很多家庭都清楚“有多少钱”，  
但并不清楚：

- 资产是否过度集中？
- 现金是否充足？
- 房产、投资、保障是否结构失衡？

本项目通过简单填写家庭资产信息，自动生成：

- 📊 **家庭资产结构全景图**
- 🧭 **结构观察、潜在风险与优化建议**
- 📄 **可导出的 PDF 家庭资产体检报告**

适合：
- 非金融、非互联网行业人群
- 想系统了解家庭资产结构的普通家庭
- 希望做家庭财务梳理但不知从何入手的用户

---

## 🚀 功能亮点

- 家庭资产结构可视化（饼图）
- 自动生成结构解读与建议（不涉及具体产品）
- 支持一键导出 PDF 报告
- 支持电脑 / 手机浏览器访问
- 无需注册、填写即可使用

---

## 🧩 使用流程

1️⃣ 填写家庭资产信息  
2️⃣ 点击「生成家庭资产全景图」  
3️⃣ 查看资产结构解读与提示  
4️⃣ 如需保存，可生成并下载 PDF 报告  

---

## 📌 适用说明与合规声明

- 本工具不涉及任何证券、基金、保险等具体产品推荐

## 📂 项目结构

- `app.py`  
  Streamlit 应用主入口，负责页面渲染与交互逻辑。

- `models.py`  
  模型规则。

- `calculator.py`  
  资产负债数据计算。

- `insights_engine.py`  
  家庭资产结构解读规则与分析逻辑。

- `pdf_report.py`  
  PDF 报告生成模板与排版逻辑。

- `requirements.txt`  
  项目运行所需的 Python 依赖列表。

- `README.md`  
  项目说明文档。
